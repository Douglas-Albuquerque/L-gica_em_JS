Lógica de Programação em Javascript

Crie um programa que leia o número de dois jogadores, crie uma lógica que sorteie um numero entre 0 e 1 caso o número sorteado for 0 o jogador que escolheu o menor numero vence e se for 1 o jogador que escolheu o maior número vence, caso os numeros escolhido por ambos os jogadores sejam iguais mostre "Empate!!".
